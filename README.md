# Alien Grover 🛸

Браузерная игра-тамагочи в Sci-Fi/Alien стилистике. Ты — исследователь на космической станции, выращиваешь фантастический инопланетный организм в био-камере: заливаешь гидро-плазму, подкармливаешь био-катализатором, включаешь LED-фитолампу и извлекаешь споры, когда рост достигнет 100%.

**Безопасный reskin-проект для публикации в Telegram Web App:** в игре полностью отсутствуют какие-либо реальные наркотические термины, образы или ассоциации — содержание выдержано в космическо-фантастической эстетике.

## Технологии
- Vanilla HTML / CSS / JS, без сборки и внешних зависимостей.
- WebAudio-синтез звуков (без ассетов).
- localStorage для сохранений (ключ `alienGroverSave`).
- Telegram Web App integration (опционально, вне Telegram работает как обычная страница).

## Запуск

```bash
python3 -m http.server 8765
```

Открой http://localhost:8765/ в браузере.

> `file://` не работает — запускать только через локальный HTTP-сервер.

## Структура
- `index.html` — разметка (HUD, селектор камер, культура, шкалы, модалки intro/lab/log, toasts).
- `style.css` — неоновый космический chunky-стиль внутри телефонного фрейма.
- `game.js` — вся логика: game loop, состояние, сохранения, события, звуки.
- `AGENTS.md` — инструкции для AI-ассистентов (включая строгий запрет наркотических терминов).

## Целевая аудитория
TWA для каталога Telegram. Контент подходит для широкой аудитории без рисков блокировок.
